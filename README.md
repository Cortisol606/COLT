# COLT — Command Launch Tool

Домашний command-palette для Windows: нажал хоткей → набрал название
команды → она выполнилась. У каждой команды может быть и своя
собственная горячая клавиша, в обход попапа. Расширяется добавлением
`.py`-файлов в папку `commands/` — без правки основного кода.

## Возможности

- Глобальный хоткей `Ctrl+Alt+Space` открывает окно поиска в любом приложении
- Нечёткий поиск по названию и алиасам команды (не нужно вводить точно)
- У любой команды можно задать свой отдельный хоткей (`HOTKEY = "ctrl+space"`),
  который выполняет её напрямую, без открытия попапа
- Новые команды подключаются простым добавлением файла в `commands/` —
  ядро трогать не нужно
- Горячая перезагрузка команд из трея, без перезапуска приложения
- Упаковывается в отдельный `.exe`, папка `commands/` при этом остаётся
  внешней и редактируемой

## Структура проекта

```
COLT/
├── main.py                 — ядро: хоткеи, окно поиска, движок команд, трей
├── requirements.txt        — зависимости
├── .gitignore
├── commands/
│   ├── sup.py               — пример: сайт + свой хоткей (Ctrl+Space)
│   ├── watch_series.py      — пример: открыть сериал по ссылке
│   ├── _template.py         — пустой шаблон для новой команды
│   └── _examples.py         — справочник всех вариаций (сайт, программа,
│                               последовательность действий, хоткей, обход UAC)
└── README.md
```

Файлы, начинающиеся с `_`, COLT не загружает как команды — это шаблоны
и справочники, безопасно лежат в папке для примера.

## Быстрый старт

### 1. Установить Python

Python 3.11+ с [python.org](https://www.python.org/downloads/).
При установке обязательно отметь **"Add python.exe to PATH"**.

Проверка:
```
python --version
```

### 2. Установить зависимости

В консоли, открытой в папке проекта:
```
pip install -r requirements.txt
```

Ставится: `PySide6` (окно), `keyboard` (глобальные хоткеи), `rapidfuzz`
(нечёткий поиск), `pystray` + `Pillow` (иконка в трее).

### 3. Запустить

```
python main.py
```

Если `keyboard` выдаёт ошибку доступа — запусти консоль **от имени
администратора** (нужно для перехвата глобальных клавиш на Windows).

Приложение уйдёт в трей. `Ctrl+Alt+Space` открывает попап, `Ctrl+Space`
сразу открывает Sup (пример команды со своим хоткеем).

## Как добавить свою команду

Любой `.py`-файл в `commands/` с таким контрактом:

```python
NAME = "Название в списке"
ALIASES = ["короткие", "варианты", "поиска"]
HOTKEY = "ctrl+shift+q"   # необязательно — своя горячая клавиша в обход попапа

def run():
    # что должно произойти при выборе команды
    ...
```

1. Скопируй `commands/_template.py` под новым именем (без `_` в начале).
2. Заполни `NAME`, `ALIASES`, при желании `HOTKEY`, и код в `run()`.
3. Сохрани файл.
4. Трей → **"Перезагрузить команды"** — новая команда сразу в поиске
   (и хоткей, если указан), без перезапуска приложения.

Больше готовых паттернов (запуск программы, последовательность из
нескольких действий, обход окна UAC через Планировщик заданий) —
в `commands/_examples.py`.

## Упаковка в .exe

```
pip install pyinstaller
pyinstaller --onedir --windowed --name COLT --uac-admin main.py
```

- `--onedir` — папка с exe вместо одного файла, запускается мгновенно
  (в отличие от `--onefile`, который каждый раз распаковывается заново)
- `--uac-admin` — exe сразу запросит права администратора, они нужны
  библиотеке `keyboard` для перехвата хоткеев

После сборки скопируй папку `commands/` внутрь `dist/COLT/`, рядом
с `COLT.exe` — команды остаются редактируемыми и без пересборки exe.

Автозапуск с Windows: ярлык на `COLT.exe` в папку `shell:startup`
(`Win + R` → `shell:startup`).

## Устранение проблем

| Проблема | Решение |
|---|---|
| Хоткей не срабатывает | Запусти от имени администратора |
| Новая команда не в поиске | Нажми "Перезагрузить команды" в трее после сохранения файла |
| Команда не выполняется | Смотри консоль/лог — печатается `[COLT] Ошибка выполнения '...'` |
| Хоткей конфликтует с другой программой | Смени `MAIN_HOTKEY` в `main.py` или `HOTKEY` в файле команды |
| Иконка в трее не появилась после сборки в exe | Пересобери с `--hidden-import pystray._win32` |
